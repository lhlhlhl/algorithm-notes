# Algorithm Notes

个人算法知识笔记本，用于记录推荐系统、生成式推荐、Semantic ID、模型评测、论文阅读和工程实践。

## 目录

- `recommendation-system/semantic-id/`：Semantic ID、codebook、tokenizer
- `recommendation-system/generative-retrieval/`：生成式召回、beam search
- `recommendation-system/evaluation/`：离线评测、实验设计、指标体系
- `paper-notes/`：论文精读笔记
- `experiment-sop/`：实验 SOP、复盘模板
